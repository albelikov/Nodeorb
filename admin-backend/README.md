# Серверная часть административной панели (Admin Backend)

## Описание
Сервис для управления административной частью логистической платформы.

## Функциональность
- Управление пользователями и ролями
- Мониторинг производительности системы  
- Статистика и аналитика операций
- Управление настройками системы
- Администрирование безопасности

## Технологии
- **Язык**: Kotlin
- **Фреймворк**: Spring Boot
- **СУБД**: PostgreSQL
- **Аутентификация**: OAuth2 + JWT

## Конфигурация
```properties
server.port=8081
spring.datasource.url=jdbc:postgresql://localhost:5432/admin_db
```

## Запуск
```bash
./gradlew bootRun

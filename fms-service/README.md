# Сервис управления флотом (Fleet Management Service - FMS)

## Описание
Система управления транспортным флотом с трекингом в реальном времени.

## Функциональность
- Мониторинг транспорта в реальном времени
- Управление водителями и сменами
- Контроль расхода топлива
- Техническое обслуживание флота
- Страхование и ПОГ
- Геймификация для водителей
- Интеграция с IoT датчиками

## Технологии
- **Язык**: Kotlin
- **Фреймворк**: Spring Boot
- **Геоданные**: PostGIS
- **IoT**: MQTT протокол
- **WebSocket**: Real-time обновления
- **СУБД**: PostgreSQL

## Конфигурация
```properties
server.port=8084
gis.integration=HERE Maps
mqtt.broker=localhost:1883
```

## Запуск
```bash
./gradlew bootRun
```

## Интеграция
- HERE Maps / OpenStreetMap
- IoT датчики транспортных средств
- ГЛОНАСС/GPS трекеры
- OBD-II системы
- TMS для маршрутизации

## Безопасность
- Аутентификация водителей
- Шифрование данных GPS
- Защита от взлома IoT
- Режим стратегической защиты

## Соответствие
- FMCSA (Federal Motor Carrier Safety Administration)
- ATA (American Trucking Association)
- Европейские транспортные стандарты

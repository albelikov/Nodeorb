# Клиентская часть административной панели (Admin Frontend)

## Описание
Фронтенд приложение для администрирования логистической платформы.

## Функциональность
- Панель мониторинга операций
- Управление пользователями и ролями
- Статистика и аналитика в реальном времени
- Настройки системы и безопасности

## Технологии
- **Кроссплатформенность**: Kotlin Multiplatform
- **Веб**: React/Vue.js
- **Нативные приложения**: Android/iOS
- **WebSocket**: для реального времени

## Сборка
```bash
./gradlew build
```

## Разработка
```bash
./gradlew run
# или для React-версии:
cd src/jsMain && npm start
```

## Архитектура
Поддерживает веб, десктоп и мобильные устройства с общим кодом на Kotlin.

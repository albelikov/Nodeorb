## Roadmap по фазам

### Phase 1 (MVP) - КРИТИЧНО

| Сущность                 | Приоритет | Описание                                  |
|--------------------------|-----------|-------------------------------------------|
| Mission                  | CRITICAL  | Глобальная задача для робота              |
| Task                     | CRITICAL  | Подзадачи миссии                          |
| NodeProfile              | CRITICAL  | Профиль робота с характеристиками        |
| MissionPlan              | CRITICAL  | План от AI с оптимизацией                 |
| MissionExecutionStatus   | CRITICAL  | Статус выполнения в реальном времени      |
| CostParameters           | CRITICAL  | Экономические параметры для оптимизации   |

### Phase 2 - ДОПОЛНЕНИЯ

| Сущность                 | Приоритет | Описание                                  |
|--------------------------|-----------|-------------------------------------------|
| WeatherCondition         | HIGH      | Погодные условия                          |
| Alert                    | HIGH      | Алерты и предупреждения                   |
| TelemetryData            | HIGH      | Телеметрия (GPS, батарея, датчики)        |
| AIDecisionLog            | HIGH      | Логи решений для XAI                      |
| MissionPerformanceMetrics| MEDIUM    | Метрики после выполнения                  |

### Phase 3 - АДВАНС

| Сущность                 | Приоритет | Описание                                  |
|--------------------------|-----------|-------------------------------------------|
| AirspaceRestriction      | MEDIUM    | Ограничения воздушного пространства       |
| TrafficData              | MEDIUM    | Данные о трафике                          |
| OptimizationScenario     | MEDIUM    | Монте-Карло симуляции                     |
| NodeAnalytics            | MEDIUM    | Аналитика по роботам                      |

### Phase 4 - ГРУППОВЫЕ МИССИИ

| Сущность                 | Приоритет | Описание                                  |
|--------------------------|-----------|-------------------------------------------|
| SwarmMission             | LOW       | Координация группы роботов                |
| A2ANegotiation           | LOW       | Переговоры между роботами за ресурсы      |
| IndustryPreset           | LOW       | Отраслевые пресеты (Agriculture, etc.)    |

## Оценка размеров таблиц БД (PostgreSQL)

| Таблица                  | Размер     | Пример использования                     |
|--------------------------|------------|-------------------------------------------|
| missions                  | Маленькая | 1000-10000 записей                       |
| tasks                     | Средняя   | 10x от missions (10 000-100 000)          |
| node_profiles             | Маленькая | 100-1000 роботов                          |
| mission_plans             | Средняя   | 10x от missions (версии планов)           |
| telemetry_data            | Огромная  | 100 записей/сек/робот (мгновенная архивация) |
| alerts                    | Средняя   | 10-1000 алертов/час                       |

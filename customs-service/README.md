# Таможенный сервис (Customs Service)

## Описание
Сервис для работы с таможенными органами и оформления таможенной документации.

## Функциональность
- Оформление таможенных деклараций
- Электронный документооборот с таможней
- Расчет таможенных платежей
- Валидация товаров и классификация ТН ВЭД
- Интеграция с ЕАЭС системами
- Генерация PDF деклараций
- Архивирование документов

## Технологии
- **Язык**: Kotlin
- **Фреймворк**: Spring Boot
- **Интеграция**: REST API + EDI
- **Документы**: PDF генерация
- **СУБД**: PostgreSQL

## Конфигурация
```properties
server.port=8083
customs.integration.endpoint=https://customs-api.gov
edi.format=EDIFACT
```

## Запуск
```bash
./gradlew bootRun
```

## Интеграция
- ACE (US Customs)
- ЕАЭС таможенные системы
- Национальные таможенные порталы

## Безопасность
- Поддержка CUI (Controlled Unclassified Information)
- Шифрование FIPS 140-2
- Полный аудит операций

## Комплаенс
Соответствует требованиям:
- Таможенное законодательство стран
- ФТС/Федеральная таможенная служба
- Цифровые стандарты документооборота

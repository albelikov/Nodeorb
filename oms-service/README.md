# Сервис управления заказами (Order Management Service - OMS)

## Описание
Центральный сервис для полного управления жизненным циклом заказов в логистике.

## Функциональность
- Создание и обработка заказов
- Расчет стоимости доставки
- Выбор оптимального маршрута
- Отслеживание статуса заказа в реальном времени
- Обработка возвратов и рекламаций
- Управление клиентской базой
- Автоматическое создание накладных

## Технологии
- **Язык**: Kotlin
- **Фреймворк**: Spring Boot
- **СУБД**: PostgreSQL
- **События**: Apache Kafka
- **Кэш**: Redis
- **Геоданные**: PostGIS

## Конфигурация
```properties
server.port=8086
kafka.bootstrap.servers=localhost:9092
database.orders.table=orders
```

## Запуск
```bash
./gradlew bootRun
```

## Интеграция
- WMS для управления складом
- TMS для планирования маршрутов
- FMS для управления флотом
- Учетные системы клиентов (SAP, Oracle)
- Системы оплаты

## Безопасность
- RBAC с ABAC для клиентских данных
- Шифрование конфиденциальной информации
- Полный аудит изменений заказа
- Режим стратегической устойчивости

## Процессы
### Стандартный заказ
1. Создание заказа → OMS
2. Резервирование запаса → WMS
3. Планирование маршрута → TMS
4. Назначение транспорта → FMS
5. Доставка и отслеживание

### Стратегический режим
При активации переопределяет обычные маршруты для приоритетных грузов.

## Нормативная база
Соответствует:
- Consumer protection laws
- Supply chain security standards
- Data privacy regulations (GDPR, CCPA)

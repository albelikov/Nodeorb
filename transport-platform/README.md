# Транспортная платформа (Transport Platform)

## Описание
Централизованная платформа для управления мультимодальными перевозками с интеграцией различных видов транспорта.

## Функциональность
- Мультимодальные перевозки комбинацией видов транспорта
- Управление договорами с перевозчиками
- Координация железнодорожных и авиаперевозок
- Электронный документооборот и подписи
- Управление тарифами и ставками
- Координация "последней мили"
- Интеграция с портовыми системами

## Технологии
- **Язык**: Kotlin
- **Фреймворк**: Spring Boot
- **Интеграция**: REST API + EDI
- **Документы**: PDF/XML генерация
- **СУБД**: PostgreSQL
- **Асинхронность**: RabbitMQ

## Конфигурация
```properties
server.port=8089
carrier.integration.endpoint=https://railway-portal.gov
signing.provider=digital-signature
```

## Запуск
```bash
./gradlew bootRun
```

## Виды транспорта
### Поддерживаемые модальности
- **Железнодорожный**: Интеграция с RailNet
- **Авиационный**: IATA стандарты
- **Морской**: Port community systems
- **Автотранспорт**: FMS интеграция
- **Смешанные**: Intermodal operations

## Электронные подписи
- Qualified electronic signatures (QES)
- Digital certificates
- Timestamping services
- Legal validity across jurisdictions

## Тарифное управление
- Dynamic pricing algorithms
- Contract management
- Rate negotiation automation
- Performance-based incentives

## Интеграция с регуляторами
- Таможенные системы (ACE, DELTA)
- Портовые сообщества (PCS)
- Авиационные системы (IATA)
- Железнодорожные диспетчеры

## Безопасность
- mTLS для интеграций
- Аудит всех операций в SCM
- Шифрование контрактов
- Контроль целостности документов

## Соответствие
- Международные транспортные конвенции
- Цифровизация цепочек поставок
- Требования электронной коммерции
- Стандарты intermodal транспорта
